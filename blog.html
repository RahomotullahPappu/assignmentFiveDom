2. How will you select html elements using DOM? Name the DOM methods.

Ans: There are five select elements

  • getElementsByTagName()

    This method is the return all elements that matches the specified Tag name

  • getElementsByClassName()

    This method is the return all elements that matches the specified Tag name

  • getElementById()

    This method is the only first element matches it and different unique id

  • querySelector()

    This method is the first element that matches the specified CSS selector in the document.

  • querySelectorAll()

    This method is returns all the elements that match the specified CSS selector in the document.
